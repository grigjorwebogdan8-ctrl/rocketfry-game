## 📋 Что в каждом файле?

### 📄 README.md
**Главный файл документации**

Содержит:
- Обзор проекта и функций
- Структуру проекта
- Список технологий
- FAQ (частые вопросы)
- Команды для запуска

**Когда использовать**: первое знакомство с проектом

---

### 📄 QUICK_START.md
**Быстрая инструкция для запуска**

Содержит:
- Команды для создания проекта
- Минимальный набор конфигов
- Краткие версии файлов
- Порядок создания файлов

**Когда использовать**: нужно быстро запустить проект

---

### 📄 EXPORT_GUIDE.md
**Подробная инструкция по экспорту**

Содержит:
- Описание проекта
- Структуру папок
- Пошаговую установку
- Примечания и советы

**Когда использовать**: нужна детальная информация об установке

---

### 📄 COMPONENTS_CODE.md
**Полный код всех компонентов**

Содержит:
- `App.tsx` (главный компонент)
- `StarBackground.tsx` (звездный фон)
- `RocketFlight.tsx` (анимация ракеты)
- `BetControls.tsx` (панель ставок)
- `BetsList.tsx` (список ставок)
- `Header.tsx` (шапка)
- `BottomNav.tsx` (навигация)
- CSS файлы

**Когда использовать**: копирование кода компонентов

---

### 📄 UI_COMPONENTS.md
**Код UI компонентов**

Содержит:
- `button.tsx` — компонент кнопки
- `input.tsx` — компонент ввода
- `utils.ts` — функция `cn()`

**Когда использовать**: создание базовых UI компонентов

---

### 📄 FILES_CHECKLIST.md
**Чек-лист всех файлов**

Содержит:
- Список всех 19 файлов

- Порядок создания файлов
- Команды для проверки

**Когда использовать**: проверка, что все файлы созданы
